# Precipitation daily concentration

Function `precipitation_concentration()` calculates daily precipitation
concentration (Martin-Vide et al. 2004).

## Usage

``` r
precipitation_concentration(p)
```

## Arguments

- p:

  A numeric vector with daily precipitation values.

## Value

Function `precipitation_concentration()` returns a value between 0
(equal distribution of rainfall) and 1 (one day concentrates all
rainfall).

## References

Martin-Vide J (2004) Spatial distribution of a daily precipitation
concentration index in peninsular Spain. International Journal of
Climatology 24, 959–971. doi:10.1002/joc.1030.
