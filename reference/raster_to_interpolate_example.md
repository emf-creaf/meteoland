# Example raster data set for interpolation of weather variables

**\[experimental\]**

Example raster data set of spatial location and topography records from
Catalonia (NE Spain). Cell size is 1km x 1km and raster size is 10x10
cells.

## Format

stars object

## Source

ICGC

## Examples

``` r
data(raster_to_interpolate_example)
```
